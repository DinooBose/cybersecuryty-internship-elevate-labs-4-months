# Linux User Management

## Commands Used

```bash
whoami
id
cat /etc/passwd
sudo adduser testuser
sudo deluser testuser
```
### Key Concepts

- Root user has full privileges

- Normal users have restricted access

- sudo allows privilege escalation safely

### Security Best Practices

- Disable unused users

- Never login directly as root

- Use sudo instead
